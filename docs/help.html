Démarrage rapide

En trois étapes, vous êtes prêt à streamer avec alertes, chat et widgets synchronisés.

  1. Téléchargez Nova Stream depuis la page d'accueil et installez l'application sur Windows.
  2. Connectez Twitch directement dans l'app (recommandé) ou via ce site, puis liez l'app au tableau de bord.
  3. Configurez vos widgets sur le site — les changements s'appliquent automatiquement dans Nova Stream si l'app est ouverte.
💡
Vous pouvez lire ce guide sans être connecté. Pour configurer vos widgets, connectez-vous avec Twitch depuis le tableau de bord.

Installation

Nova Stream est une application desktop Windows. Aucune installation serveur n'est nécessaire de votre côté.

  1. Rendez-vous sur la page d'accueil ou sur GitHub Releases pour télécharger la dernière version.
  2. Exécutez le fichier Nova-Stream-Setup-x.x.x.exe et suivez l'assistant d'installation.
  3. Au premier lancement, acceptez les autorisations réseau si Windows vous le demande (liaison avec le site et Twitch).
  4. Ouvrez Paramètres dans l'app pour vérifier le dossier d'enregistrement et les options audio.
ℹ️
L'application se met à jour automatiquement lorsqu'une nouvelle version est publiée, si Nova Stream est ouverte au moment de la release.

Connexion Twitch

La connexion Twitch active le chat, les alertes EventSub (follow, sub, raid, bits) et la récupération automatique de votre clé stream.

Recommandé

Option A — Dans l'application

Onglet Apps → Se connecter avec Twitch. Une fenêtre OAuth s'ouvre dans votre navigateur ; validez, puis revenez à l'app.

Option B — Via le site web

Connectez-vous sur ce site avec Twitch, ouvrez Nova Stream sur votre PC, puis utilisez Lier à Nova Stream depuis le tableau de bord.

Après la connexion

  • Votre pseudo et avatar apparaissent dans la barre latérale du tableau de bord.
  • La clé stream RTMP peut être récupérée automatiquement (Paramètres → Stream).
  • Les intégrations Twitch (chat, alertes, objectifs) sont disponibles dans l'app et sur le site.

Liaison site ↔ application

Le site sert de panneau de configuration à distance. La liaison envoie vos réglages et votre session Twitch à Nova Stream sur votre PC.

  1. Lancez Nova Stream sur votre ordinateur (l'app doit rester ouverte).
  2. Connectez-vous sur ce site et ouvrez le tableau de bord.
  3. Descendez jusqu'à la section Lier à Nova Stream — le statut doit passer au vert si l'app est détectée.
  4. Cliquez sur Lier à Nova Stream. La première liaison transmet aussi votre compte Twitch à l'application.
⚠️
La liaison utilise le port local 3847. Si l'app n'est pas détectée, vérifiez que Nova Stream est lancé et que le pare-feu Windows autorise les connexions entrantes.

Synchronisation automatique

Une fois lié, chaque modification sur le site (widgets, alertes, dons) est envoyée à l'app en environ une demi-seconde, sans recliquer sur « Lier ».

Première diffusion

Nova Stream gère la capture, le mixage audio et l'envoi vers Twitch.

  1. Ajoutez vos sources (écran, fenêtre, webcam, micro) dans la scène active.
  2. Dans Paramètres → Stream, vérifiez la clé Twitch (récupérée automatiquement si connecté).
  3. Définissez le titre et la catégorie via Go Live avant de passer en direct.
  4. Activez l'enregistrement local dans Paramètres → Général si vous souhaitez aussi enregistrer sur disque.
  5. Cliquez sur Diffuser pour lancer le live, ou Enregistrer pour une session locale uniquement.

Widgets & alertes

Chaque widget se configure depuis le tableau de bord ou sa page dédiée. Un aperçu en direct vous montre le rendu avant diffusion.

Widget Rôle Configuration
🔔 Fenêtre d'alerte Follow, abonnement, raid, bits, don — avec son et animation personnalisables. Configurer
💬 Fenêtre de chat Affichage du chat Twitch sur scène (style, nombre de messages). Configurer
🎯 Objectif followers Barre de progression vers un objectif de followers. Configurer
⭐ Objectif abonnés Suivi des abonnements en direct sur votre scène. Configurer
👁 Spectateurs Compteur de viewers en temps réel. Configurer
📊 Sondage Question et options affichées pendant le live. Configurer
💰 Dons Page de tip publique et alertes après paiement confirmé. Configurer

Types d'alertes Twitch

  • Follow — nouveau follower
  • Abonnement — sub et resub
  • Raid — arrivée d'un autre streamer
  • Bits / Cheers — dons de bits Twitch
  • Don — paiement PayPal via votre page de tip

OBS & overlays

Nova Stream intègre nativement les widgets dans ses scènes. Vous pouvez aussi utiliser les URLs overlay dans OBS Studio.

  1. Ouvrez la page de configuration du widget souhaité sur ce site.
  2. Copiez l'URL overlay fournie (elle contient un token unique).
  3. Dans OBS : Sources → Navigateur (Browser Source) → collez l'URL.
  4. Ajustez largeur et hauteur selon l'aperçu du widget (souvent 800×600 ou selon le widget).
⚠️
Les données live (chat, viewers, alertes) nécessitent que Nova Stream soit lancé sur votre PC. Un overlay OBS vide indique souvent une app fermée ou une URL expirée.

Dons PayPal

Recevez des pourboires via une page publique, avec alerte en direct dans Nova Stream après confirmation du paiement.

  1. Allez sur la page Dons et activez les dons.
  2. Connectez votre compte PayPal (Standard ou Business).
  3. Copiez votre lien public du type tip.html?u=votre_pseudo et partagez-le (chat, panneau Twitch, bio).
  4. Personnalisez montants suggérés, devise, message et GIF d'alerte si besoin.
  5. L'alerte live n'apparaît qu'après paiement confirmé par PayPal — pas à la simple ouverture de la page.

Mises à jour

Nova Stream évolue régulièrement. Voici comment rester informé.

  • Application — mise à jour automatique au lancement ou en arrière-plan si l'app est ouverte.
  • Site web — déployé automatiquement à chaque modification ; aucune action de votre part.
  • Discord — annonces dans le salon #mises-a-jour (releases app et déploiements site).

Dépannage

Solutions aux problèmes les plus fréquents.

L'application n'est pas détectée lors de la liaison

Vérifiez que Nova Stream est ouvert avant de cliquer « Lier ». Autorisez l'app dans le pare-feu Windows (port 3847). Redémarrez l'application si le statut reste gris.

Chat ou alertes inactifs

Déconnectez puis reconnectez Twitch dans l'onglet Apps de l'application. Vérifiez que la liaison site ↔ app est active sur le tableau de bord.

Session web expirée

Reconnectez-vous sur le site avec Twitch. La session est conservée environ 4 heures, puis une nouvelle connexion est demandée.

Overlay OBS vide ou figé

Lancez Nova Stream, vérifiez l'URL overlay (token valide) et actualisez la source navigateur dans OBS (clic droit → Actualiser).

Les widgets ne se synchronisent pas

Assurez-vous que l'app est liée et ouverte. Modifiez un réglage sur le site et attendez une seconde — le statut de sync apparaît dans la barre latérale du tableau de bord.

Don reçu mais pas d'alerte

Seuls les paiements confirmés déclenchent l'alerte. Vérifiez l'historique sur la page Dons et que PayPal est bien connecté.
