📖 Guide utilisateur

Tout ce qu'il faut pour installer Nova Stream, connecter Twitch, configurer widgets et dons.

Installation

  1. Téléchargez Nova Stream depuis la page d'accueil ou GitHub Releases.
  2. Installez Nova-Stream-Setup-x.x.x.exe et lancez l'application.
  3. Connectez Twitch (voir ci-dessous).
  4. Configurez votre clé stream dans Paramètres (récupération auto si Twitch connecté).
  5. Activez l'enregistrement local dans Paramètres → Général si besoin, puis Diffuser ou Enregistrer.

Connexion Twitch

Option A — Directement dans l'app (recommandé)

Onglet Apps → Se connecter avec Twitch. Une fenêtre OAuth s'ouvre dans le navigateur.

Option B — Via le site web

  1. Connectez-vous sur ce site avec Twitch.
  2. Lancez Nova Stream sur votre PC.
  3. Tableau de bord → Lier à Nova Stream.

Widgets & alertes

  • Configurez chaque widget depuis le tableau de bord.
  • Les modifications se synchronisent automatiquement si l'app est ouverte.
  • URL overlay web pour OBS : copiez l'URL depuis la page de configuration du widget.
  • Types d'alertes : follow, sub, raid, bits, don — sons personnalisables par type.

Dons PayPal

  1. Page Dons → activez les dons.
  2. Connectez PayPal Standard ou Business.
  3. Partagez votre lien public tip.html?u=votre_pseudo.
  4. L'alerte live n'apparaît qu'après paiement confirmé.

Dépannage

  • App non détectée — Lancez Nova Stream avant de cliquer « Lier ». Vérifiez le pare-feu Windows (port 3847).
  • Chat / alertes inactifs — Reconnectez Twitch. Déconnectez puis reconnectez dans Apps.
  • Session web expirée — Reconnectez-vous sur le site (session conservée ~4 h).
  • Overlay OBS vide — L'app doit tourner pour les données live ; vérifiez l'URL avec token.
