# Nova Stream — Guide utilisateur

Nova Stream est **gratuit et utilisable par tout le monde**. Chaque personne installe l'app sur son PC et connecte **son propre compte Twitch**.

## Comment ça fonctionne ?

```
Vous (utilisateur A)          Vous (utilisateur B)
      │                              │
      ▼                              ▼
  Nova Stream sur votre PC     Nova Stream sur son PC
  Vos tokens, vos scènes       Ses tokens, ses scènes
  Stockage 100% local          Stockage 100% local
```

Comme OBS ou Streamlabs Desktop : **pas de serveur central obligatoire**. Vos données restent sur votre machine.

## Installation (utilisateur final)

1. Téléchargez **Nova-Stream-Setup.exe** depuis [GitHub Releases](https://github.com/Hazeyy5/nova-stream/releases)
2. Installez et lancez l'application
3. Connectez-vous avec Twitch (site web ou onglet Apps)
4. Configurez votre clé de stream dans Paramètres
5. Cliquez **Diffuser en direct**

## Connexion Twitch (2 options)

### Option A — Via le site (recommandé)
1. Ouvrez [hazeyy5.github.io/nova-stream](https://hazeyy5.github.io/nova-stream)
2. Connectez-vous avec Twitch
3. Lancez Nova Stream sur votre PC
4. Cliquez **Lier à Nova Stream**

### Option B — Directement dans l'app
1. Onglet **Apps** → **Se connecter avec Twitch**

## Questions fréquentes

**Dois-je créer une app Twitch ?**  
Non. C'est le mainteneur du projet qui configure l'app Twitch officielle « Nova Stream » une seule fois. Tous les utilisateurs passent par celle-ci.

**Mes données sont-elles partagées avec d'autres utilisateurs ?**  
Non. Chaque installation est indépendante.

**Puis-je utiliser Nova Stream sans compte Twitch ?**  
Oui, pour enregistrer localement ou streamer vers une autre plateforme avec une URL RTMP personnalisée.
